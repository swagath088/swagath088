### 🏥 Hospital Management System
**Full-Stack Web Application**

A full-stack Mini Hospital Management System with a Django REST Framework backend and a React.js + React Router frontend. Patients register/login, view doctor availability, book/cancel appointments, and track appointment history. Doctors manage their availability and view booked appointments — each from a dedicated dashboard. Race-condition-safe booking via `transaction.atomic()` + `select_for_update()`, Google Calendar sync via OAuth2, and a decoupled serverless email microservice for notifications.

**Full-Stack Web Application**
